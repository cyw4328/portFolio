Content

개인의 정보를 공유할 수 있는 블로그 사이트 입니다.

회원은 개인 블로그에 게시글을 작성하여 정보를 저장할 수 있으며,
업로드 된 게시글은 공개범위에 따라 전체보기인 것은 다른 이용자들이 볼 수 있으며, 서로 댓글과 대댓글로 소통할 수 있습니다.

메인페이지에서는 공감수가 높은 순으로 보여주는 부분과 최신 순으로 보여지는 영역이 있으며, 각 카테고리 별 해당하는 개인의 메뉴안에 있는 게시글을 리스트로 보여줍니다.

개인의 정보를 확인 수정 할 수 있는 마이페이지 기능과 관리자 아이디 접속 시 사용 할 수 있는 관리자 기능을 분리 시켜놓았습니다.

1차 프로젝트라 기본에 충실하여 프로젝트를 진행하였습니다.

Responsibilities

1) 팀장 업무 :
프로젝트 전반적인 일정 관리 및 담당 업무 배치, 코드 관리 업무
2) 검색 기능 :
사이트 전체 회원의 게시 글을 분류에 따라 검색할 수 있는 기능 구현
3) 리스트 불러오기 기능 :
모든 회원의 게시 글 중 공개 범위가 전체 보기 인
게시 글을 불러와 나열해주는 기능 구현
4) 카테고리 , 메뉴 기능 :
대 분류 , 중 분류 , 개인 메뉴 기능 추가 및 삭제 기능 구현
( 카테고리에 속한 메뉴에 있는 게시 글을 불러오거나 블라인드 처리 )
5) 신고 기능 :
게시 글을 신고 항목 별로 나누어 신고 처리 할 수 있는 기능 구현
6) MY PAGE 기능 :
내 정보를 수정 기능 , 구독 관계 수정 기능 , 알림 기능 구현
내가 적은 게시 글 및 댓글 관리 기능 구현
